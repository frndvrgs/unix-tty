06
Rules
the invariants
→ do
use monospace for all text, always
keep backgrounds in the screen-black family
let phosphor glow be the only decoration
use scanlines as texture, never as spectacle
make scanlines and flicker user-toggleable
pick one theme per layout
let silence and spacing carry weight
x do not
use pure #000000 as background
add gradients, shadows, or blur
use rounded corners on anything
add colors outside the palette
mix green and amber in the same layout
use proportional or sans-serif fonts
animate with bounce or spring easing
